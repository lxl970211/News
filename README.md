#News
News是一款基于安卓平台开发的新闻客户端软件，软件实现新闻分类查看、新闻搜索、新闻评论、收藏、注册、登录等功能，并对新闻数据进行了缓存<br>
##项目
项目使用Xuils框架进行开发，项目中的API来自于聚合数据，新闻评论和收藏功能自建服务器进行实现使用Mysql数据库进行数据保存<br>
新闻服务器端地址：https://github.com/lxl970211/news-server<br>
